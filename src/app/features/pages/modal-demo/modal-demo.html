Reusable Modal Component

One <app-modal> component — three different patterns. Content is projected via #modalBody and #modalFooter ng-template slots.

ℹ️

Info Modal

Simple message with no footer. Close via ✕, backdrop click, or Esc.

size: sm
⚠️

Confirm Modal

Confirm / Cancel footer buttons with result feedback.

size: md @if (confirmResult()) {

{{ confirmResult() }}

}
📝

Form Modal

NgModel two-way binding keeps form state in the component.

size: lg @if (formSubmitted()) {

{{ formResult() }}

}

app-modal is a standalone reusable component that lives in core/components/modal. Drop it in any feature and project your own content.

  • Angular 17+ input() / output() signals
  • Slots via ng-template #modalBody and #modalFooter
  • Size variants: sm · md · lg
  • Closes on backdrop, ✕ button, or Esc
