Data Techniques in Use

  • @Output Step 1 emits validated data to parent
  • @Input Parent passes data down to Step 2 for pre-filling
  • @Output Step 2 emits updated data + back event to parent
  • Service + Signal Step 3 reads directly from FormDataService.formStateSignal
